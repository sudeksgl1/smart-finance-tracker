# 💰 Smart Finance Tracker

Python, Streamlit ve SQLite kullanılarak geliştirilmiş kişisel finans takip uygulaması.

## Özellikler
- Gelir / gider takibi
- Sabit gelir ve gider yönetimi
- Grafikler ve analizler
- Finansal uyarılar
- Soft / Dark / Light tema
- Emoji animasyonları

## Kullanılan Teknolojiler
- Python
- Streamlit
- SQLite
- Pandas

## Çalıştırma

```bash
pip install -r requirements.txt
streamlit run app.py
```
